# /audit-ui-360

1. Activa el sub-agente de navegador integrado.
2. Navega a la página de detalles del vehículo (VDP) en un viewport de dispositivo móvil (390x844).
3. Interactúa con el visor de imágenes 360° arrastrando el cursor/pantalla.
4. Haz clic en al menos dos "Hotspots" (marcadores de daño o características de lujo).
5. Verifica que las ventanas emergentes (tooltips/modales) no se desborden de la pantalla y toma una captura de pantalla (screenshot) para validación visual.
