# SEO and Data Structure

Implementa estrictamente **Schema Markup (JSON-LD)** para vehículos (`Car` o `Vehicle`) en todas las páginas de detalles del vehículo (VDP).

Convenciones de nomenclatura de base de datos: usa **snake_case** (ej. `vehicle_inventory`, `trade_in_leads`).

Seguridad: **TODAS** las consultas a la base de datos deben estar parametrizadas para evitar inyecciones SQL.

Valida siempre los formatos del **Número de Identificación del Vehículo (VIN)** en el backend y frontend antes de procesar cualquier dato.
